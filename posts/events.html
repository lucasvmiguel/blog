Events in Node.js

In this first post I'll talk quickly about events in Node.js. First of all, I'll explain what is events in any programming language.

A event is a signal, someone emit a signal and someone listen this signal(could be more than one listener). In Node.js, the signal is able to pass some parameters and who are listen can treat this parameters as you wish.

Pay attention to comments...

  
  //add the event module to the application
  var events = require('events');
  //this object do the hard work
  var eventEmitter = new events.EventEmitter();

  //The method that listen('on') - receive a string(event name) and a function(receive the event parameter)
  eventEmitter.on('eventDispatch', function(variable){
    console.log('Receive the event with parameter: ' + variable);
  });

  //This method emit the event with a string(event name) and anything(event parameter)
  eventEmitter.emit('eventDispatch', 'Lucas');

It's important to realize that you need listen the event before being fired!

For more info: https://nodejs.org/api/events.html
